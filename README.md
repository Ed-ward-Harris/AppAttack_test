# AppAttack Toolkit

## Installation

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/Ed-ward-Harris/appAttack_test.git
   cd appAttack_test
   ```
2. **Run the install script as sudo**:
```bash
   sudo ./install.sh
```
3. **Start the Application: Once installed, you can run the toolkit using**:
```bash
   appAttack_toolkit
```
